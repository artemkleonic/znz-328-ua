Як відкрити сайт

1. Розпакуйте архів.
2. Відкрийте файл index.html подвійним кліком.
3. Усі сторінки, стилі, скрипти, зображення та частина документів уже лежать всередині папки.
4. Для розміщення на хостингу завантажте вміст папки на сервер.

Структура:
- index.html — головна сторінка
- assets/styles.css — стилі
- assets/app.js — пошук документів і мобільне меню
- assets/img — оптимізовані зображення
- assets/docs — офіційні документи
